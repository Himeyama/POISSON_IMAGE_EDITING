# POISSON_IMAGE_EDITING

# HOW TO INSTALL

```
apt install git -y
apt install libopencv-dev
apt install uuid-dev
```

## BUILD

```bash
git clone https://github.com/hikaright/POISSON_IMAGE_EDITING
cd POISSON_IMAGE_EDITING
make
sudo make install
```
